Recursion

Adding up an array of numbers without a loop

One Challenge Iâ€™ve enjoyed recently was adding numbers in an array with recursion. I was taking a Javascript recursion class and the instructor was teaching us a few ways to use recursion. He gave us students a few recursion challenges and one that stood out to me is doing addition in recursion while only passing one value into the method. After some thought the solution I came up with was to remove the first value in the array and add it to the next item then call the method again. It's really a simple solution. Here is the code in javascript 

var sumArray = function (arr) {
	if (arr.length === 1) {
		return arr[0]
	} else {
		var firstNum = arr.shift()
		arr[0] += firstNum
		return sumArray(arr)
	}
};

It's a very simple solution though from what i read its not the most efficient since solving the problem with a regular loop would be faster and less resource intensive but it's really fun to solve some recoursion challenges and this is a really simple one to learn.
