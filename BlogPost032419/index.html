The first one on my list is called max_by. This method can be called on an array of objects and then it would sort the array in alphabetical or numerical order based on the object property of your choice. Then it returns the highest value or last string of the sorted array.

Here is an example to illustrate what max_by does.

[“Dario”, “Cantaloupe”, “Apple”].max_by{ |object| object.length}

Here we pass an array of words and it max_by will return “Cantaloupe” The array of three words was sorted by the length and we got back the longest word

If we ran the line of code with just the object without asking for the length like this

[“Dario”, “Cantaloupe”, “Apple”].max_by{ |object| object}

We would get back “Dario” since it has the last letter if the array was sorted alphabetically. But if we are just sorting by string names or just numbers alone then we could use the method called max like this

[“Dario”, “Cantaloupe”, “Apple”].max()

And it would also return “Dario” since it works in the same way but without taking a block.
