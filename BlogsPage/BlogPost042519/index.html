Getting started with P5

042519

There are plenty of javascript libraries to choose from they each have their own use with their own pros and cons. Today I want to talk about p5. P5 is based around the idea being easy to pick up. Its not the most powerful javascript library but its very quick to get started and making some animations.

Here are some examples here to show some of what P5 can do. https://p5js.org/examples/

Setting up P5 is like any other JS library to include it into your project you just gotta link it in your header. Either include it the file locally or place a link to the CDN you can find either link here https://p5js.org/get-started/

Then in your JS file you just need a setup and update method in normal javascript.

function setup() {

}

function draw() {

}

The setup method is run once at the start of the program and the update method is run continuously. In the update is where the main parts of your program will go.

P5 brings plenty of methods for you to use. Most of them are simplified to single words like the ellipse and rect methods where you just need to say the method name, size, and location on screen. You can see all the included functions on their site here https://p5js.org/reference/ You can also use plenty of extra libraries for p5 to do things like interacting with the Dom or adding other kinds of inputs instead of the default. https://p5js.org/libraries/ They also have a webeditor where you can test out a few methods quickly https://editor.p5js.org/
