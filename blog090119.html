What I like about python so far on my first day with it

Recently i decided to pickup Python. While studying at Flatiron we learned Ruby and javascript, I enjoyed using Them but it seems like the job market for ruby was not that big and Javascript felt like a chore to write in. I've read that python was better for code Challenges which i felt like i need to improve on and the job market for python was much larger. So this week I started studying some python and its been going great. I feel like the syntax is simple to learn, looks very clean, and is easy to remember.

If statements and loops look so great and after writing a lot of Ruby and Javascript i enjoy the reduced about of keys needed to write python.

In Ruby

if i < 10
	puts ‘less than ten’
end

in javascript

if ( i < 10){
	console.log(‘less than ten’)
}

And python

if i < 10:
	print ‘less than ten’

No end statement need for python! It knows your done with the if statement because the change of indentation! The same applies to functions loops and everything else. It makes the code much easier to read and when solving challenges. I will use it for future algo themed blog posts and projects
