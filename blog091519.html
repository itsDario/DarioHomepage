Try and Except

handling errors in python

I had another code challenge. The challenge was simple. Return true if the string was a number and false if the string contained anything that was not a number. I chose Python as the language since I was learning it and found it fun so far.

At first I tried to use an if statement on a float() method to test if the string was a number but python does not work like that. After a quick google search I learn about try and except. They are used like if statements but to test if a piece of code does not throw any errors. Try is used for code that might return an error and except is used for hold code that runs if the try does return an error than it will run the code in the except section.

This was my resulting code

try:
	float(num)
	return True
except:
	return False

The code runs the code in the try section if it fails then runs the code in the except section really easy to use. If you want to run seprate code when a spicific error happens then you can do something like

try:
	Code
except NameError:
	Other code
except :
	Last code

With this code you can make more stable programs even if the program relies on external information like form data. Good luck with your future challenges!
