Css animations with keyframes

041119

CSS animations are a simple and fun way to add a bit of life to your website. They are quick to implement and today i'll be covering how to use keyframes to help you make any animation. There are two parts to an animations. The animation properties and the keyframes.

Examples:

https://jsfiddle.net/itsDario/09z4ntow/29/

Here are a few animation properties you can use to put into your css

animation-name: example;

This sets how long your animation plays

animation-duration: 4s;

This sets how many times you play your animation. It can be set to Infinite

animation-iteration-count: 5;

This is how long you want the animation to wait before starting up It also accepts negative numbers which makes the animation start midway through

animation-delay: 2s;

There are a few more animations properties you can learn about here

https://www.w3schools.com/css/css3_animations.asp

With keyframes you enter three things. The percent into the animation you want the event to occur, the property you want changed and the value of the property

@keyframes example {

0% {background-color:red;}

50% {background-color:blue;}

100% {background-color:red;}

}

This animation will move the background back and forth between red and blue. Feel free to play around the the settings in the jsFiddle! I wish you luck with your future CSS animations
